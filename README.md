# SafeGate

This project aims to help small businesses implement data security internal controls by proxying local access to HTTPS data and using locally deployed LLM analytics.

## Prototype

### Stage.1

Now the development is in stage.1, which is to build a C proxy for assignment. This part will eliminate after entering Satge.3.

### Stage.2

Stage.2 is to build prototype level app.

### Stage.3

Stage.3 is to build a real product named "SafeGate", to help small company to check the upstreaming data to make sure the company's data is not leaking, also check the downstreaming data to make sure there is no phishing or melware.
